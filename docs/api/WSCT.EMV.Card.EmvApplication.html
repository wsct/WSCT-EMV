Class EmvApplication

An EMVApplication instance represents an EMV application. Allows selection and use of the application.

TlvSignedDynamicApplicationResponse

Accessor to the Signed Dynamic Application rAPDU obtained by INTERNAL AUTHENTICATE.

Declaration
public TlvData TlvSignedDynamicApplicationResponse { get; }
Property Value
Type Description
WSCT.Helpers.BasicEncodingRules.TlvData
Remarks

From EMV book 3:

Format 1: The data object returned in the rAPDU message is a primitive data object with tag equal to '80'. The value field consists of the value field of the Signed Dynamic Application Data as specified in Book 2

Format 2: The data object returned in the rAPDU message is a constructed data object with tag equal to '77'. The value field may contain several BER-TLV coded objects, but shall always include the Signed Dynamic Application Data as specified in Book 2.
