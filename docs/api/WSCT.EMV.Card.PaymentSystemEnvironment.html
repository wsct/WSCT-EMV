Class PaymentSystemEnvironment

A PaymentSystemEnvironment instance represents the PSE of an EMV Card. Allows selection of the PSE and reading the associated EF.

FCI, and records are stored when obtained in WSCT.Helpers.BasicEncodingRules.TlvData format.

Remarks

By default, "1PAY.SYS.DDF01" is used for the name of the PSE

Examples
ICardChannel cardChannel;
// ... acquire here a valid cardChannel instance ...
PaymentSystemEnvironment pse = new EMVApplication(cardChannel);
pse.name = "1PAY.SYS.DDF01"; // optional, as this value is set by default
pse.select();
pse.read();
